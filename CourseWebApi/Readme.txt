This project contains the final source code for the course web api application
You'll need:
1. Visual Studio 2013 Ultimate
2. SQL Server 2014 Express or Developer (or higher)
Setups:
1)	Create a database called Course in SQL Server 2014
2)	Run the Table scripts for Course and Prerequisite table , Sql scripts can be found at \\CourseWebApi\Course.Persistence\Sql folder
3)	The database implementation can be turned off by setting the EnablePersistence app setting key to False. 
4)	The Web API can be tested by running Unit Tests or any rest client testing tool provided by google chrome or Firefox
